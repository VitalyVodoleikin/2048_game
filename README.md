# 🎮 2048 — классическая головоломка на Python

> «Простое управление, но бесконечная тяга сделать ещё один ход»

Моя реализация культовой игры **2048** на Python с использованием библиотеки `pygame`. Чистый код, понятная архитектура.
Есть возможность расширить функционал.

---

## ✨ Особенности проекта

- ✅ Классическая механика 2048: сдвиг, объединение, случайное появление плитки.
- 🧠 Минималистичный и читаемый код — удобно изучать и модифицировать.
- 🛠 Легко кастомизировать: цвета, размеры, логику, счётчики.
- 📦 Зависимости собраны в понятном виде.
- 🐍 Python 3 + pygame — кроссплатформенность (Windows/macOS/Linux).

---

## !!! Реализовано

- классическая механика 2048: сдвиг, объединение, случайное появление плитки;
- реализация графического отображения: цвета, размеры, логика, счётчики;
- запрос имени игрока с внесением в базу данных;
- список рекордов с привязкой к базе данных;
- сохранение результата при выходе из игры в файл и возможность продолжить игру с сохраненного места;
- конвертация игры в один исполняемый .exe-файл;
- тестирование ряда функций для корректной работы.
